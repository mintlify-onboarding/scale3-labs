---
title: Slack
description: Steps to setup alerts for Slack
---

1. Select **Connect Workspace** at https://www.scale3labs.com/autopilot/settings/integrations

![Slack](/images/slack.png)

2. Select the channel you want to add the **scale3-intel** bot to post node update and network announcements too

![Slack](/images/slack-permissions.png)

3. Click **Allow**

![Slack](/images/slack-allow.png)

4. Done!
